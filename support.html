一、应用简介与适用场景

FamilyMedVault 面向家庭场景下的健康信息整理，包括但不限于：医疗记录与附件、用药与计划提醒、症状记录与趋势、预约日历（可与系统日历协同）、就医协调类手记（如转诊进度由您自行更新）、紧急卡片导出等。 具体功能以您设备上安装的版本为准。

二、系统与环境要求

  • 平台：iOS（通过 Apple App Store 分发）。
  • 网络：核心记录功能以本地优先为主；部分可选能力（例如云端 AI、自建备份地址访问）需要互联网连接。
  • 更新：建议保持 iOS 与应用为较新版本，以获得安全修复与兼容性改进。

三、订阅与账单（Apple App Store）

会员权益（若适用）通过 App 内购买 / 自动续期订阅 由 Apple 计费。 我们无法在应用外为您修改 Apple 账单或退款；订阅的开通、变更与取消通常在 Apple 账户侧完成。

3.1 查看或取消订阅（常规路径）

  1. 打开 iPhone 「设置」 → 轻点顶部 Apple ID（姓名）。
  2. 进入 「订阅」。
  3. 在列表中选择 FamilyMedVault（或显示名称以 App Store 为准）。
  4. 在此查看续订日期、方案与取消订阅选项。

取消订阅后，通常可在当前账单周期结束前继续使用已付费权益；具体以 Apple 界面说明为准。

3.2 恢复购买（Restore）

若您已在当前 Apple ID 下购买过符合条件的权益，可在应用内会员相关页面使用恢复购买（若版本提供该入口）。 仍无法识别时，请确认登录的是购买时所使用的 Apple ID，并检查网络与系统时间。

3.3 退款与账单争议

如需申请退款或咨询扣款问题，请通过 Apple 官方支持渠道提交请求；开发者无法代替 Apple 批准退款。

四、权限说明（为何需要）

  • 通知：用于用药、预约等本地提醒（若您启用）。若关闭通知权限，提醒可能无法弹出。
  • 日历（读/写）：仅在您启用相关同步或写入功能时使用，用于与系统日历协同。
  • 相机 / 相册：用于添加影像附件或 OCR 辅助录入（若您主动操作）。
  • 面容 ID / 触控 ID（若您启用应用锁）：用于解锁应用，数据留在设备侧处理。

您可随时在 iOS 设置 → 隐私与安全性 中查看或撤回权限；撤回可能导致对应功能不可用。

五、功能相关常见问题（FAQ）

5.1 首页 /「当前成员」与预约列表

  • 提示先选择当前成员：请在应用中选定当前家庭成员后再添加记录或预约；从「家庭」返回「日历」等页面后，若列表未刷新，可切换离开该标签页再进入一次。
  • 预约保存在哪里：预约条目显示在日历页下方「当前成员的预约」列表中；请向下滑动查看。

5.2 预约日历与系统日历

  • 双向同步：若开启同步，新建或更新的预约可能写入本机日历账户；您在「日历」App 中修改时间后，可使用应用内「从日历拉回更新」尝试同步回应用（需日历权限）。
  • 删除预约：在预约条目卡片上可使用「删除预约」；若曾写入系统日历，应用将尽力移除对应日程（受系统与权限影响）。

5.3 提醒与「安排 / 重新安排」

  • 提醒不出现：请检查系统通知权限、请勿打扰模式、以及预约开始时间与提前分钟数是否合理（提醒触发时间须在未来）。
  • 点击无反应或提示无法安排：请查看弹窗说明；常见原因包括通知权限被拒绝、提醒时间已过期、或系统拒绝调度。

5.4 用药与随访等功能

  • 用药计划与提醒依赖本地通知；修改计划后请关注下一次提醒时间是否合理。
  • 症状趋势等功能可能对会员等级有要求，请以应用内提示为准。

5.5 协调 / 转诊手记

  • 相关状态由您手动维护，用于个人备忘与跟进；不连接医院或保险公司的实时系统。

5.6 云端 AI（若您启用）

  • 需在隐私设置中满足相应同意条件；云端处理可能依赖您配置的地址与服务可用性。
  • 输出仅供参考，不能替代执业医师判断。

5.7 加密云备份（若您启用）

  • 备份目的地由您配置的 URL / 凭据决定；请确保服务器可信、传输使用 HTTPS，并妥善保管令牌。
  • 运营者不默认托管您的备份服务器。

5.8 通过 Apple 登录（若可用）

  • 登录依赖 Apple ID；若您配置了后端校验地址，请确保该地址正确且服务可用。

5.9 导出与删除数据

  • 导出：若设置中提供数据导出，导出文件可能保存在本机「文件」或应用文档目录；请自行妥善保管导出文件。
  • 删除：应用内删除操作通常清除本应用在本机存储的数据；此操作可能不可恢复，请谨慎确认。

六、故障排查（通用步骤）

  1. 重启应用；必要时重启设备。
  2. 将 iOS 与 FamilyMedVault 更新至最新版本。
  3. 确认网络畅通（使用云端功能时）。
  4. 检查相关系统权限是否开启。
  5. 尝试退出并通过 Apple ID 恢复购买（仅订阅相关问题）。
  6. 若问题仍存在，请邮件联系我们，并附上：设备型号、iOS 版本、应用版本、复现步骤与截图。
